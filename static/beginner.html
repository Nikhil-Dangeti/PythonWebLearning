Print Statement in Python

The print() function is used in Python to display output on the screen (terminal). It helps us print numbers, text, or multiple values easily.

How to Print Different Types of Values?

Printing a Number: Just give the number inside print(): 
print(6)

Output: 6

Printing a Word (String): Use double quotes " " or single quotes ' ' for text: 
print("nani")
print('hello')

Output:
nani
hello

Printing a Single Character: We can print a single letter inside quotes: 
print('a')

Output: a

Printing Multiple Values Together: We can print different types of values together by separating them with commas: 
print(1, 2, "ram")

Output: 1 2 ram
(The values are separated by spaces by default.)

print(10)
print("Python is easy!")
print("Number:", 25)

Output:
10

Conditional Statements in Python

Conditional statements are used to make decisions in Python. They allow the program to execute different code blocks based on conditions.

1. Types of Conditional Statements

a) if Statement

The if statement checks a condition. If the condition is True, it executes the code inside the if block.

Syntax: 

if condition:
    # Code to execute if condition is True
                
age = 18
if age >= 18:
    print("You are eligible to vote.")

Output: You are eligible to vote.

b) if-else Statement

The if-else statement allows the program to execute one block if the condition is True and another block if it is False.

Syntax: 

if condition:
    # Code if condition is True
else:
    # Code if condition is False
                
num = 10
if num % 2 == 0:
    print("Even number")
else:
    print("Odd number")

Output: Even number

c) if-elif-else Statement

The if-elif-else statement is used when we have multiple conditions to check.

Syntax: 

if condition1:
    # Code if condition1 is True
elif condition2:
    # Code if condition2 is True
else:
    # Code if no conditions are True
                
marks = 85
if marks >= 90:
    print("Grade: A")
elif marks >= 80:
    print("Grade: B")
elif marks >= 70:
    print("Grade: C")
else:
    print("Grade: D")

Output: Grade: B

d) Nested if Statements

We can place an if statement inside another if statement, called nested if. 

num = 15
if num > 10:
    print("Number is greater than 10")
    if num % 2 == 0:
        print("It is an even number")
    else:
        print("It is an odd number")

Output:
Number is greater than 10
It is an odd number

Example 1: Check if a number is positive, negative, or zero

Indentation in Python

In Python, indentation is very important. It refers to the spaces at the beginning of a code line. Unlike many other programming languages that use curly braces { } to define code blocks, Python uses indentation to define blocks of code.

Proper indentation is necessary for Python to understand the structure of the code. If the indentation is not consistent, Python will raise an IndentationError.

Rules of Indentation:

  • All statements within the same block must be indented at the same level.
  • Typically, 4 spaces are used for each indentation level (although tabs can be used, spaces are recommended).
  • Mixing tabs and spaces in indentation should be avoided.

Example: Correct Indentation

age = 20
if age >= 18:
    print("You are an adult.")
    print("You can vote.")

Output:
You are an adult.
You can vote.

Example: Incorrect Indentation

age = 20
if age >= 18:
print("You are an adult.")
print("You can vote.")

Error:
IndentationError: expected an indented block

String Operations in Python (Detailed Explanation)

A string in Python is a sequence of characters enclosed in single ('), double ("), or triple (''' or """) quotes. 

s1 = 'Hello'
s2 = "Python"
s3 = '''Welcome to Python'''

1. String Operations in Python
Python provides various operations to manipulate strings, such as concatenation, slicing, repetition, searching, and modification.

A) String Concatenation (+ Operator)

Concatenation means joining two or more strings using the + operator. 

str1 = "Hello"
str2 = "World"
result = str1 + " " + str2
print(result)

Output: Hello World

B) String Repetition (* Operator)

Repeats a string multiple times. 

word = "Python "
print(word * 3)

Output: Python Python Python

C) String Slicing ([] Operator)

Extracts a portion of a string. Indexing starts from 0 in Python. 

text = "Hello, Python!"
print(text[0:5])   # Prints 'Hello'
print(text[:5])    # Same as above
print(text[7:])    # Prints 'Python!'
print(text[-7:])   # Prints 'Python!'

Output:
Hello
Hello
Python!
Python!

D) String Length (len() Function)

Finds the length (number of characters) in a string. 

text = "Python"
print(len(text))

Output: 6

E) String Membership (in and not in Operators)

Checks whether a substring is present in a string. 

text = "Welcome to Python"
print("Python" in text)      # True
print("Java" not in text)    # True

F) Converting Case (upper(), lower(), title(), capitalize())

These methods change the case of letters in a string. 

text = "Hello Python"

print(text.upper())    # HELLO PYTHON
print(text.lower())    # hello python
print(text.title())    # Hello Python
print(text.capitalize())  # Hello python

G) Removing Spaces (strip(), lstrip(), rstrip())

Removes extra spaces from a string. 

text = "  Hello Python  "
print(text.strip())   # "Hello Python" (removes spaces from both sides)
print(text.lstrip())  # "Hello Python  " (removes left spaces)
print(text.rstrip())  # "  Hello Python" (removes right spaces)

H) String Replacement (replace())

Replaces a part of a string with another string. 

text = "I love Java"
new_text = text.replace("Java", "Python")
print(new_text)

Output: I love Python

I) Splitting and Joining Strings (split(), join())

split() breaks a string into a list of words. join() joins a list of words into a single string. 

text = "apple,banana,grapes"
words = text.split(",")  # Splits string into a list
print(words)

sentence = " ".join(words)  # Joins list into a string
print(sentence)

Output:
['apple', 'banana', 'grapes']
apple banana grapes

J) Finding Substrings (find(), index())

find() returns the position of the first occurrence of a substring (-1 if not found). index() is similar but raises an error if the substring is not found. 

text = "I love Python"
print(text.find("Python"))  # 7
print(text.find("Java"))    # -1 (not found)
print(text.index("Python")) # 7

K) Checking String Type (isalpha(), isdigit(), isalnum())

isalpha() ‚Üí Returns True if all characters are letters.
isdigit() ‚Üí Returns True if all characters are numbers.
isalnum() ‚Üí Returns True if the string has letters and numbers only. 

text1 = "Python"
text2 = "1234"
text3 = "Python123"

print(text1.isalpha())  # True
print(text2.isdigit())  # True
print(text3.isalnum())  # True

print("Length of text:", len(text))

print("Substring:", text[0:6])   # Python

Loops in Python (Detailed Explanation)

Loops are used to execute a block of code multiple times. Python provides two main types of loops:

for loop ‚Äì Used for iterating over sequences like lists, tuples, strings, etc.
while loop ‚Äì Used when the number of iterations is unknown and depends on a condition.

1. for Loop
The for loop is used to iterate over sequences (lists, tuples, strings, dictionaries, sets, etc.).

Syntax: 

for variable in sequence:
    # Code to execute in each iteration

Example 1: Iterating over a List

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

This code iterates through each item in the list fruits and prints it one by one.

Output:
apple
banana
cherry

Example 2: Using range() in a for loop

The range(start, stop, step) function generates a sequence of numbers. 

for i in range(1, 6):  # Numbers from 1 to 5
    print(i)

Here, range(1, 6) generates numbers from 1 to 5, and each number is printed in the loop.

Output:
1
2
3
4
5

Example 3: Looping Through a String

word = "Python"
for char in word:
    print(char)

The code iterates through each character of the string word and prints each character on a new line.

Output:
P
y
t
h
o
n

Example 4: Using else with for Loop

The else block runs after the loop finishes successfully. 

for i in range(3):
    print("Iteration:", i)
else:
    print("Loop finished successfully!")

After completing all iterations, the else block is executed, printing a final message.

Output:
Iteration: 0
Iteration: 1
Iteration: 2
Loop finished successfully!

2. while Loop
The while loop runs as long as a given condition is True.

Syntax: 

while condition:
    # Code to execute

Example 1: Basic while Loop

count = 1
while count <= 5:
    print("Count:", count)
    count += 1

The loop prints numbers starting from 1 up to 5 by incrementing the count by 1 in each iteration.

Output:
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

Example 2: Using else with while Loop

The else block executes when the loop condition becomes False. 

x = 3
while x > 0:
    print("X:", x)
    x -= 1
else:
    print("Loop finished")

After the condition x > 0 becomes false, the else block executes and prints a message.

Output:
X: 3
X: 2
X: 1
Loop finished

Example 3: Infinite Loop (with break)

A while loop runs forever if the condition is always True. We can use break to stop it. 

while True:
    print("This is an infinite loop")
    break

Although True would create an infinite loop, the break statement immediately exits the loop after the first iteration.

Output: This is an infinite loop

3. Loop Control Statements

A) break Statement

Stops the loop immediately when a condition is met. 

for num in range(1, 6):
    if num == 3:
        break  # Loop stops when num is 3
    print(num)

The loop prints numbers 1 and 2. When num equals 3, the break statement stops the loop.

Output:
1
2

B) continue Statement

Skips the current iteration and moves to the next one. 

for num in range(1, 6):
    if num == 3:
        continue  # Skips when num is 3
    print(num)

When num equals 3, the continue statement skips printing and moves to the next number.

Output:
1
2
4
5

C) pass Statement

Used as a placeholder when a loop is required but no code needs to be executed. 

for num in range(5):
    pass  # Does nothing

The pass statement means "do nothing." It allows the code to run without errors even though no action is taken inside the loop.

4. Nested Loops
A loop inside another loop is called a nested loop. 

for i in range(1, 4):
    for j in range(1, 4):
        print(i, j)

The outer loop (i) runs 3 times, and for each iteration of i, the inner loop (j) also runs 3 times, printing all pairs.

Output:
1 1
1 2
1 3
2 1
2 2
2 3
3 1
3 2
3 3

5. Example Codes for Loops

Example 1: Sum of First n Natural Numbers

n = int(input("Enter a number: "))
sum = 0
for i in range(1, n + 1):
    sum += i
print("Sum:", sum)

This program calculates the sum of all natural numbers from 1 to n using a for loop.

Output:
Enter a number: 5
Sum: 15

Example 2: Factorial of a Number Using while Loop

num = int(input("Enter a number: "))
factorial = 1
while num > 0:
    factorial *= num
    num -= 1
print("Factorial:", factorial)

The code calculates the factorial by multiplying numbers from num down to 1 using a while loop.

Output:
Enter a number: 5
Factorial: 120

Mini Project 2: Password Strength Checker

In this project, we will create a program that checks the strength of a password entered by the user. The program will use loops, if-else statements, and string methods to validate the password. It will also allow the user to re-enter the password if it is invalid and limit the number of attempts.

Rules for a Strong Password:

  • At least 6 characters long.
  • Must include uppercase and lowercase letters.
  • Must include at least one digit.
  • Must include at least one special character (!, @, #, etc.).
  • Should not contain spaces.

Maximum Attempts: 3 tries allowed to enter a valid password.

Tips for creating a strong password:

  • Use a mix of uppercase and lowercase letters.
  • Include numbers and special characters.
  • Avoid common words or easily guessable information.

    elif (has_upper or has_lower) and has_digit:
        print("‚ö° Moderate Password: Try adding special characters for more strength.\n")
        attempt += 1
    else:
        print("‚ö†Ô∏è Weak Password: Add uppercase letters, digits, and special characters.\n")
        attempt += 1

if attempt == max_attempts:
    print("\nüö´ Maximum attempts reached! Please try again later.")

Game Rules:

  • Rock beats Scissors
  • Scissors beats Paper
  • Paper beats Rock
  • If both players choose the same, it‚Äôs a tie

computer_move = "paper"

computer_move = "scissors"

print("Invalid move! Please enter rock, paper, or scissors.")
else:
print(f"Computer chose: {computer_move}")
print(f"You chose: {user_move}")

# Determine the winner
if user_move == computer_move:
    print("It's a tie!")
elif (user_move == "rock" and computer_move == "scissors") or \
        (user_move == "scissors" and computer_move == "paper") or \
        (user_move == "paper" and computer_move == "rock"):
    print("You win! üéâ")
else:
    print("Computer wins! üòé")

Sample Output:

Explanation:
The computer uses the random.random() function to pick a number between 0 and 1. Logical operators like and and comparison operators like > and < are used to map this number to "rock", "paper", or "scissors". Then the user's input is compared against the computer's choice using if-elif-else to determine the winner.
