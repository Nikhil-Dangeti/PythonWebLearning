Beginner Python Quiz

1. What type of loop is used when the number of iterations is known?

a) while loop
b) for loop
c) Infinite loop
d) do-while loop

2. What will be the output of the following code?

for i in range(3):
    print(i)
else: 
    print("Loop finished!")
a) 0 1 2
b) 0 1 2 Loop finished!
c) 0 1 2 3 Loop finished!
d) Infinite loop

3. Which statement is used to skip the current iteration and continue with the next one?

a) break
b) continue
c) pass
d) skip

4. What will be the output of the following code?

count = 1
while count < 4:   
    print(count)   
    count += 1
a) 1 2 3
b) 1 2 3 4
c) 0 1 2 3
d) Infinite loop

5. What happens if the condition in a while loop is always True and there is no break statement?

a) Loop executes once
b) Syntax error
c) Infinite loop
d) The loop will not execute

6. How do you correctly use a for loop to iterate over a string?

a) for char in "Python":
b) for i in range("Python"):
c) for i in "Python".range():
d) for i in range(len("Python")):

7. What does the pass statement do in a loop?

a) Skips the rest of the loop
b) Ends the loop
c) Does nothing (acts as a placeholder)
d) Skips the current iteration

8. What is the output of the following code?

for i in range(1, 5, 2):    
    print(i)
a) 1 2 3 4
b) 1 3
c) 2 4
d) 1 2 3

9. How many times will the loop run?

for i in range(3, 10, 2):    
    print(i)
a) 3 times
b) 4 times
c) 5 times
d) 6 times

10. What is the correct syntax for a while loop?

a) while (i < 5) {}
b) while i < 5:
c) while i < 5
d) loop while i < 5:

11. Which loop executes at least once, even if the condition is false?

a) for loop
b) while loop
c) do-while loop
d) None of the above

12. Which of the following is used to iterate over a dictionary?

a) for key in dict:
b) for key in dict.keys():
c) for key, value in dict.items():
d) All of the above

13. What will be the output of this nested loop?

for i in range(2):  
    for j in range(2):  
        print(i, j)
a) (0,0) (1,1)
b) (0,0) (0,1) (1,0) (1,1)
c) (0,0) (0,1)
d) (1,0) (1,1)

14. What will the following code print?

for i in range(3):
    if i == 1:       
        break   
    print(i)
a) 0 1 2
b) 0
c) 1
d) No output

15. What will happen if break is used inside a while loop?

a) Terminates the loop immediately
b) Skips the next iteration
c) Skips the rest of the code in the current iteration
d) Throws an error
