Intermediate Python Quiz

1. What will be the output of the following code?

def multiply(a, b=2):
    return a * b

print(multiply(3))
a) 3
b) 6
c) 9
d) Error

2. What does the following lambda function do?

cube = lambda x: x ** 3
print(cube(2))
a) Returns 6
b) Returns 8
c) Returns 9
d) Returns 4

3. What will be the output of the following code?

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

print(factorial(4))
a) 24
b) 12
c) 6
d) 8

4. What will be the output of the following list operation?

numbers = [1, 2, 3, 4, 5]
numbers[1:3] = [10, 20]
print(numbers)
a) [1, 10, 20, 4, 5]
b) [1, 2, 3, 10, 20]
c) [10, 20, 3, 4, 5]
d) [1, 10, 20, 3, 4, 5]

5. What will be the output of the following code?

numbers = [1, 2, 3, 4, 5]
print(numbers[-2:])
a) [4, 5]
b) [3, 4]
c) [5, 4]
d) [1, 2]

6. What will be the output of the following tuple operation?

tup = (1, 2, 3) + (4, 5)
print(tup)
a) (1, 2, 3, 4, 5)
b) (1, 2, 3, 4)
c) (1, 2, 3, 5)
d) Error

7. What will be the output of the following code?

tup = (10, 20, 30, 40)
print(tup[1:3])
a) (10, 20)
b) (20, 30)
c) (30, 40)
d) (10, 20, 30)

8. What will be the output of the following set operation?

set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}
print(set1.symmetric_difference(set2))
a) {1, 2, 5, 6}
b) {3, 4}
c) {1, 2, 3, 4, 5, 6}
d) {1, 2}

9. What will be the output of the following code?

numbers = {1, 2, 3, 4, 5}
numbers.discard(3)
print(numbers)
a) {1, 2, 4, 5}
b) {1, 2, 3, 4, 5}
c) {3}
d) Error

10. What will be the output of the following dictionary operation?

d = {"a": 1, "b": 2}
d.update({"c": 3})
print(d)
a) {'a': 1, 'b': 2, 'c': 3}
b) {'a': 1, 'b': 2}
c) {'c': 3}
d) Error

11. What will be the output of the following code?

d = {"a": 1, "b": 2, "c": 3}
print(d.get("b"))
a) 1
b) 2
c) 3
d) None

12. What will be the output of the following dictionary comprehension?

d = {x: x*2 for x in range(1, 4)}
print(d)
a) {1: 2, 2: 4, 3: 6}
b) {1: 1, 2: 2, 3: 3}
c) {1: 3, 2: 6, 3: 9}
d) Error

13. What will be the output of the following nested dictionary access?

d = {"student": {"name": "Alice", "age": 21}}
print(d["student"]["age"])
a) Alice
b) 21
c) {"name": "Alice", "age": 21}
d) Error

14. What will be the output of the following code using defaultdict?

from collections import defaultdict
d = defaultdict(int)
d["a"] += 1
print(d["a"])
a) 0
b) 1
c) 2
d) Error

def sum_numbers(*args):
    return sum(args)

print(sum_numbers(1, 2, 3, 4))
a) 10
b) 6
c) 8
d) Error
