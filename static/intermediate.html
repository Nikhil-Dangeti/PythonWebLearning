Intermediate Python

Functions in Python

Functions are reusable blocks of code that perform a specific task. They help in code organization, modularity, and reusability, making programs more efficient and easier to maintain.

1. Types of Functions in Python

a) Built-in Functions

Python provides many predefined functions that perform common tasks. Examples include:

  • print() – Displays output on the screen.
  • len() – Returns the length of an object.
  • type() – Returns the data type of a variable.
  • sum() – Adds elements of an iterable.

Example:

print(len("Python"))  # Length of the string "Python"
print(type(10))  # Returns type of the variable (int)
print(sum([1, 2, 3, 4]))  # Sums the numbers in the list

Output:
6
<class 'int'>
10

len("Python") returns 6 because "Python" has six characters.
type(10) returns <class 'int'> indicating 10 is an integer.
sum([1, 2, 3, 4]) returns 10 because 1+2+3+4 = 10.

b) User-Defined Functions

A function defined by the user to perform a specific task.

Syntax:

def function_name(parameters):
    # Function body
    return value  # (optional)

Example:

def greet(name):
    print("Hello,", name)

greet("Nikhil")

Output:
Hello, Nikhil

🔹 The function greet(name) takes name as a parameter and prints a greeting message.
🔹 Calling greet("Nikhil") outputs "Hello, Nikhil".

c) Function with Parameters and Return Value

Functions can take parameters (inputs) and return values.

Example:

def add_numbers(a, b):
    return a + b

result = add_numbers(5, 10)
print("Sum:", result)

Output:
Sum: 15

The function add_numbers(a, b) adds two numbers and returns the result.
The value 15 is stored in result and printed.

d) Default Parameters

A function parameter can have a default value, which is used if no argument is provided.

Example:

def greet(name="User"):
    print("Hello,", name)

greet("Alice")  
greet()  # Uses default value

Output: Hello, Alice
Hello, User

If no argument is given, name takes the default value "User".
Calling greet("Alice") prints "Hello, Alice", while greet() prints "Hello, User".

e) Variable-Length Arguments (*args, **kwargs)

Python allows functions to accept a variable number of arguments.

  • *args – Used for multiple positional arguments.
  • **kwargs – Used for multiple keyword arguments.

Example:

def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3, 4, 5))  # Multiple arguments

Output:
15

  • The *numbers argument allows passing multiple values to sum_all().
  • The function sums all values: 1+2+3+4+5 = 15.

f) Lambda Functions (Anonymous Functions)

A lambda function is a short, one-line function without a name.

Syntax:

lambda arguments: expression

Example:

square = lambda x: x * x
print(square(5))

Output:
25

  • The lambda function lambda x: x * x squares the input number.
  • Calling square(5) returns 25.

g) Recursive Functions

A function that calls itself to solve a problem recursively.

Example:

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))

Output:
20

  • The function factorial(n) calls itself with n-1 until n == 0
  • The result is 5 * 4 * 3 * 2 * 1 = 120.

Example Codes for Functions

Example 1: Function to Find Maximum of Three Numbers

def find_max(a, b, c):
    if a > b and a > c:
        return a
    elif b > c:
        return b
    else:
        return c

print("Max:", find_max(10, 25, 15))

Output:
Max: 25

  • The function find_max(a, b, c) compares three numbers and returns the largest.
  • It checks if a is the largest, then b, and finally c.
  • In find_max(10, 25, 15), 25 is the largest, so it is returned.

Example 2: Function to Check if a Number is Even or Odd

def check_even_odd(num):
    if num % 2 == 0:
        return "Even"
    else:
        return "Odd"

print(check_even_odd(7))
print(check_even_odd(12))

Output:
Odd
Even

  • The function check_even_odd(num) checks if a number is divisible by 2.
  • If num % 2 == 0, it returns "Even", otherwise "Odd"
  • check_even_odd(7) returns "Odd", while check_even_odd(12) returns "Even".

Lists in Python

A list is a collection of ordered, mutable, and heterogeneous elements in Python. Lists allow us to store multiple items in a single variable and can hold different data types (integers, strings, floats, etc.).

✅ Lists are:
✔ Ordered – Elements have a defined sequence.
✔ Mutable – We can modify lists (add, remove, update elements).
✔ Heterogeneous – Can store different data types in the same list.
✔ Indexed – Access elements using index numbers (starting from 0).

1. Creating Lists

We create lists using square brackets [ ] or the list() function.

Example:

# Creating lists
numbers = [1, 2, 3, 4, 5]  
fruits = ["apple", "banana", "cherry"]  
mixed = [10, "Python", 3.14]  

# Using list() function
empty_list = list()

print(numbers)  
print(fruits)  
print(mixed)  
print(empty_list)

Output:
[1, 2, 3, 4, 5]
['apple', 'banana', 'cherry']
[10, 'Python', 3.14]
[]

  • numbers contains integers.
  • fruits contains strings.
  • mixed contains an integer, a string, and a float.
  • empty_list is an empty list.

2. Accessing List Elements

We use indexing and slicing to access elements in a list.

a) Indexing (Positive & Negative Indexing)

Indexing starts from 0 (left to right) and -1 (right to left).

Example:

fruits = ["apple", "banana", "cherry", "orange"]
print(fruits[0])   # First element
print(fruits[2])   # Third element
print(fruits[-1])  # Last element

Output:
apple
cherry
orange

  • fruits[0] returns "apple".
  • fruits[2] returns "cherry".
  • fruits[-1] returns "orange" (last element).

b) Slicing a List

Slicing allows us to get a sublist using [start:end:step].

Example:

numbers = [10, 20, 30, 40, 50, 60, 70]
print(numbers[1:4])   # Elements from index 1 to 3
print(numbers[:3])    # First three elements
print(numbers[3:])    # From index 3 to end
print(numbers[::2])   # Every second element
print(numbers[::-1])  # Reverse the list

Output:
[20, 30, 40]
[10, 20, 30]
[40, 50, 60, 70]
[10, 30, 50, 70]
[70, 60, 50, 40, 30, 20, 10]

  • numbers[1:4] gets elements from index 1 to 3.
  • numbers[:3] gets the first three elements.
  • numbers[::-1] reverses the list.

3. List Operations

a) Adding Elements to a List

We can add elements using:
1️⃣ append() – Adds element at the end.
2️⃣ insert() – Adds element at a specific index.
3️⃣ extend() – Merges another list.

Example:

fruits = ["apple", "banana"]
fruits.append("cherry")  # Adds "cherry" at the end
fruits.insert(1, "orange")  # Inserts "orange" at index 1
more_fruits = ["grape", "mango"]
fruits.extend(more_fruits)  # Extends list with more_fruits

print(fruits)

Output:
['apple', 'orange', 'banana', 'cherry', 'grape', 'mango']

  • append("cherry") adds "cherry" at the end.
  • insert(1, "orange") adds "orange" at index 1.
  • extend(more_fruits) merges the two lists.

b) Removing Elements from a List

We can remove elements using:
1️⃣ remove(value) – Removes first occurrence of a value.
2️⃣ pop(index) – Removes element at the given index (default: last).
3️⃣ del list[index] – Deletes element at index.
4️⃣ clear() – Empties the list.

Example:

fruits = ["apple", "banana", "cherry", "orange"]
fruits.remove("banana")  # Removes "banana"
fruits.pop(1)  # Removes element at index 1 ("cherry")
del fruits[0]  # Deletes first element ("apple")
fruits.clear()  # Removes all elements

print(fruits)

Output:

[]

🔹 remove("banana") deletes "banana".
🔹 pop(1) removes "cherry".
🔹 clear() removes all elements.

c) Updating List Elements

We can change elements using indexing.

Example:

fruits = ["apple", "banana", "cherry"]
fruits[1] = "orange"  # Changes "banana" to "orange"
print(fruits)

Output:

['apple', 'orange', 'cherry']

🔹 fruits[1] = "orange" updates index 1 value.

4. Common List Methods

MethodDescription
append(x)Adds x to the end.
insert(i, x)Inserts x at index i.
extend(iterable)Merges another list.
remove(x)Removes first occurrence of x.
pop(i)Removes item at index i.
clear()Removes all elements.
index(x)Returns index of x.
count(x)Counts occurrences of x.
sort()Sorts list in ascending order.
reverse()Reverses list order.

Example Codes for Lists

Example 1: Find the Largest Number in a List

numbers = [10, 25, 5, 40, 15]

max_value = max(numbers)  # Finds max value
print("Largest number:", max_value)

Output:

Largest number: 40

🔹 The max() function finds the largest value.

Example 2: Remove Duplicates from a List

numbers = [10, 20, 10, 30, 20, 40]
unique_numbers = list(set(numbers))  # Convert to set and back to list
print("Unique numbers:", unique_numbers)

Output:

Unique numbers: [40, 10, 20, 30]

🔹 The set() removes duplicates.
🔹 Converting back to list() retains unique values.

Tuples in Python

A tuple is a collection of ordered, immutable, and heterogeneous elements in Python. Tuples are similar to lists but cannot be changed once created. They are useful for storing fixed data that should not be modified.

✅ Tuples are:
✔ Ordered – Elements maintain a defined sequence.
✔ Immutable – Cannot be changed after creation (no adding/removing elements).
✔ Heterogeneous – Can store different data types.
✔ Indexed – Access elements using an index (starting from 0).

1. Creating Tuples

Tuples are created using parentheses ( ) or the tuple() function.

Example:

# Creating tuples
numbers = (1, 2, 3, 4, 5)  
fruits = ("apple", "banana", "cherry")  
mixed = (10, "Python", 3.14)  

# Single element tuple (needs a comma)
single_element = (5,)  

# Using tuple() function
empty_tuple = tuple()

print(numbers)  
print(fruits)  
print(mixed)  
print(single_element)  
print(empty_tuple)

Output:

(1, 2, 3, 4, 5)
('apple', 'banana', 'cherry')
(10, 'Python', 3.14)
(5,)
()

🔹 single_element = (5,) – A comma is required for a single-element tuple.
🔹 empty_tuple = tuple() – Creates an empty tuple.

2. Accessing Tuple Elements

We use indexing and slicing to access tuple elements.

a) Indexing (Positive & Negative Indexing)

Tuples support both positive (left to right) and negative (right to left) indexing.

Example:

fruits = ("apple", "banana", "cherry", "orange")
print(fruits[0])   # First element
print(fruits[2])   # Third element
print(fruits[-1])  # Last element

Output:

apple
cherry
orange

🔹 fruits[0] returns "apple".
🔹 fruits[2] returns "cherry".
🔹 fruits[-1] returns "orange" (last element).

b) Slicing a Tuple

We can extract a sub-tuple using slicing [start:end:step].

Example:

numbers = (10, 20, 30, 40, 50, 60, 70)
print(numbers[1:4])   # Elements from index 1 to 3
print(numbers[:3])    # First three elements
print(numbers[3:])    # From index 3 to end
print(numbers[::2])   # Every second element
print(numbers[::-1])  # Reverse the tuple

Output:

(20, 30, 40)
(10, 20, 30)
(40, 50, 60, 70)
(10, 30, 50, 70)
(70, 60, 50, 40, 30, 20, 10)

🔹 numbers[::-1] reverses the tuple.

3. Tuple Operations

a) Concatenation & Repetition

We can join and repeat tuples using + and *.

Example:

tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)

new_tuple = tuple1 + tuple2  # Concatenation
repeated_tuple = tuple1 * 3  # Repetition

print(new_tuple)
print(repeated_tuple)

Output:

(1, 2, 3, 4, 5, 6)
(1, 2, 3, 1, 2, 3, 1, 2, 3)

🔹 tuple1 + tuple2 joins the two tuples.
🔹 tuple1 * 3 repeats the tuple 3 times.

b) Checking Element Existence

We use the in keyword to check if an element exists in a tuple.

Example:

fruits = ("apple", "banana", "cherry")
print("banana" in fruits)  # True
print("mango" in fruits)   # False

Output:

True
False

🔹 "banana" in fruits returns True.

4. Common Tuple Methods

Since tuples are immutable, they have limited methods compared to lists.

MethodDescription
count(x)Counts occurrences of x in the tuple.
index(x)Returns the index of x.

Example:

numbers = (10, 20, 30, 10, 40, 10)
print(numbers.count(10))   # Count occurrences of 10
print(numbers.index(30))   # Find index of 30

Output:

3
2

🔹 count(10) returns 3 because 10 appears three times.
🔹 index(30) returns 2, which is the index of 30.

5. Tuple Packing & Unpacking

a) Tuple Packing

Packing means storing multiple values in a tuple.

Example:

person = ("John", 25, "Engineer")
print(person)

Output:

('John', 25, 'Engineer')

🔹 The person tuple stores three values.

b) Tuple Unpacking

We can extract values from a tuple into variables.

Example:

person = ("John", 25, "Engineer")
name, age, profession = person  # Unpacking
print(name)
print(age)
print(profession)

Output:

John
25
Engineer

🔹 name, age, and profession store values from person.

Example Codes for Tuples

Example 1: Swapping Two Variables Using Tuples

a = 10
b = 20

a, b = b, a  # Swapping values

print("a =", a)
print("b =", b)

Output:

a = 20
b = 10

🔹 Tuple swapping makes swapping easy.

Example 2: Returning Multiple Values from a Function

def get_dimensions():
    width = 15
    height = 10
    return width, height  # Returning a tuple

dimensions = get_dimensions()
print(dimensions)
print("Width:", dimensions[0])
print("Height:", dimensions[1])

Output:

(15, 10)
Width: 15
Height: 10

🔹 Functions return multiple values as tuples.

Key Takeaways

✅ Tuples are ordered, immutable, and heterogeneous.
✅ We use indexing and slicing to access elements.
✅ Concatenation & repetition allow modifying tuple contents indirectly.
✅ Tuple packing & unpacking simplify variable assignments.

Sets in Python

A set is an unordered, mutable, and unindexed collection of unique elements in Python. Sets are useful when you need to store distinct values and perform mathematical set operations like union, intersection, and difference.

✅ Sets are:
✔ Unordered – Elements have no fixed order.
✔ Mutable – We can add or remove elements.
✔ Unique – No duplicate values.
✔ Unindexed – Elements cannot be accessed by index.

1. Creating Sets

Sets are created using curly braces {} or the set() function.

Example:

# Creating sets
numbers = {1, 2, 3, 4, 5}
fruits = {"apple", "banana", "cherry"}
mixed = {10, "Python", 3.14}

# Using set() function
empty_set = set()  # Creates an empty set (not {} because {} creates a dictionary)

print(numbers)
print(fruits)
print(mixed)
print(empty_set)

Output:

{1, 2, 3, 4, 5}
{'banana', 'cherry', 'apple'}
{10, 'Python', 3.14}
set()

🔹 Order is not guaranteed in sets.
🔹 {} creates a dictionary, so use set() for an empty set.

2. Accessing Set Elements

Since sets are unordered and unindexed, elements cannot be accessed using indexing or slicing. Instead, we check for membership using the in keyword.

Example:

fruits = {"apple", "banana", "cherry"}
print("banana" in fruits)  # True
print("mango" in fruits)   # False

Output:

True
False

🔹 We cannot access elements by index like fruits[0].

3. Modifying Sets

a) Adding Elements

We use add() to insert a single element and update() to add multiple elements.

Example:

numbers = {1, 2, 3}
numbers.add(4)  # Adding a single element
numbers.update([5, 6, 7])  # Adding multiple elements

print(numbers)

Output:

{1, 2, 3, 4, 5, 6, 7}

🔹 add() adds a single element.
🔹 update() adds multiple elements from a list/tuple.

b) Removing Elements

We use remove(), discard(), or pop() to remove elements.

Example:

numbers = {1, 2, 3, 4, 5}

numbers.remove(3)  # Removes 3 (error if not found)
numbers.discard(6) # No error if 6 is not found
removed_element = numbers.pop()  # Removes a random element

print(numbers)
print("Removed:", removed_element)

Output (may vary):

{2, 4, 5}
Removed: 1

🔹 remove(x) raises an error if x is not found.
🔹 discard(x) does not raise an error if x is missing.
🔹 pop() removes a random element.

4. Set Operations

Python provides built-in operations for union, intersection, difference, and symmetric difference.

a) Union (| or union())

Combines all elements from both sets (no duplicates).

Example:

set1 = {1, 2, 3}
set2 = {3, 4, 5}

union_set = set1 | set2  # Using | operator
# union_set = set1.union(set2)  # Alternative way

print(union_set)

Output:

{1, 2, 3, 4, 5}

🔹 Combines elements from both sets.

b) Intersection (& or intersection())

Finds common elements between two sets.

Example:

set1 = {1, 2, 3}
set2 = {3, 4, 5}

intersection_set = set1 & set2  # Using & operator
# intersection_set = set1.intersection(set2)  # Alternative way

print(intersection_set)

Output:

{3}

🔹 Only common elements are included.

c) Difference (- or difference())

Finds elements present in one set but not in another.

Example:

set1 = {1, 2, 3}
set2 = {3, 4, 5}

difference_set = set1 - set2  # Using - operator
# difference_set = set1.difference(set2)  # Alternative way

print(difference_set)

Output:

{1, 2}

🔹 Elements only in set1, not in set2.

d) Symmetric Difference (^ or symmetric_difference())

Finds elements not common in both sets.

Example:

set1 = {1, 2, 3}
set2 = {3, 4, 5}

sym_diff_set = set1 ^ set2  # Using ^ operator
# sym_diff_set = set1.symmetric_difference(set2)  # Alternative way

print(sym_diff_set)

Output:

{1, 2, 4, 5}

🔹 Excludes common elements (3).

5. Set Methods

MethodDescription
add(x)Adds an element x to the set.
update(iterable)Adds multiple elements from an iterable.
remove(x)Removes x (raises an error if x is missing).
discard(x)Removes x (no error if x is missing).
pop()Removes and returns a random element.
clear()Removes all elements from the set.
union(set2)Returns a new set with elements from both sets.
intersection(set2)Returns a new set with common elements.
difference(set2)Returns a new set with elements in set1 but not set2.
symmetric_difference(set2)Returns a set with elements in either but not both.

6. Example Applications of Sets

Example 1: Removing Duplicates from a List

numbers = [1, 2, 3, 3, 4, 4, 5]
unique_numbers = set(numbers)  # Convert list to set

print(unique_numbers)

Output:

{1, 2, 3, 4, 5}

🔹 Sets automatically remove duplicates.

Example 2: Finding Common Elements in Two Lists

list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]

common_elements = set(list1) & set(list2)  # Using intersection

print(common_elements)

Output:

{3, 4}

🔹 Fast way to find common elements.

Dictionaries in Python 📝

A dictionary in Python is an unordered, mutable, and indexed collection of key-value pairs. It is one of the most powerful and widely used data structures in Python. Dictionaries allow fast lookups, modifications, and deletions of values based on their unique keys.

✅ Features of Dictionaries
✔ Unordered – The items do not maintain any specific order (Python 3.6+ preserves insertion order).
✔ Mutable – You can add, modify, or remove elements.
✔ Indexed by keys – Instead of numerical indexes (like lists/tuples), dictionaries use unique keys to access values.
✔ Heterogeneous – Can store different data types as values.
✔ No duplicate keys – Each key must be unique, but values can be duplicated.

1. Creating a Dictionary

Dictionaries are created using curly braces {} or the dict() constructor.

Example:

# Creating a dictionary
student = {
    "name": "Alice",
    "age": 21,
    "grade": "A",
    "courses": ["Math", "Physics"],
}

print(student)

Output:

{'name': 'Alice', 'age': 21, 'grade': 'A', 'courses': ['Math', 'Physics']}

🔹 Keys ("name", "age", "grade", "courses") must be unique and immutable (strings, numbers, tuples).
🔹 Values ("Alice", 21, "A", ["Math", "Physics"]) can be of any data type.

2. Accessing Dictionary Elements

We can access values using keys with square brackets [] or the get() method.

Example:

student = {"name": "Alice", "age": 21, "grade": "A"}

# Using square brackets []
print(student["name"])  # Alice

# Using get() method (avoids KeyError if key is missing)
print(student.get("age"))  # 21
print(student.get("gender", "Not specified"))  # Default value

Output:

Alice
21
Not specified

🔹 student["name"] returns "Alice".
🔹 student.get("gender", "Not specified") returns "Not specified" because "gender" does not exist.

3. Modifying Dictionary Elements

We can update values using keys and add new key-value pairs.

Example:

student = {"name": "Alice", "age": 21, "grade": "A"}

# Updating existing value
student["age"] = 22  

# Adding a new key-value pair
student["city"] = "New York"

print(student)

Output:

{'name': 'Alice', 'age': 22, 'grade': 'A', 'city': 'New York'}

🔹 student["age"] = 22 updates "age" from 21 to 22.
🔹 student["city"] = "New York" adds a new entry.

4. Dictionary Methods

a) keys(), values(), and items()

These methods retrieve keys, values, and key-value pairs.

Example:

student = {"name": "Alice", "age": 21, "grade": "A"}

print(student.keys())    # dict_keys(['name', 'age', 'grade'])
print(student.values())  # dict_values(['Alice', 21, 'A'])
print(student.items())   # dict_items([('name', 'Alice'), ('age', 21), ('grade', 'A')])

🔹 keys() returns a list-like object of keys.
🔹 values() returns a list-like object of values.
🔹 items() returns key-value pairs as tuples.

b) update() – Merging Dictionaries

We can update an existing dictionary with another dictionary.

Example:

student = {"name": "Alice", "age": 21}
extra_info = {"grade": "A", "city": "New York"}

student.update(extra_info)
print(student)

Output:

{'name': 'Alice', 'age': 21, 'grade': 'A', 'city': 'New York'}

🔹 update() merges two dictionaries.
🔹 If keys overlap, values from extra_info replace existing ones.

c) pop() & popitem() – Removing Elements

We can remove elements using pop() (specific key) or popitem() (last inserted item).

Example:

student = {"name": "Alice", "age": 21, "grade": "A"}

# Removing a specific key
age = student.pop("age")
print(age)  # 21

# Removing the last inserted item
last_item = student.popitem()
print(last_item)  # ('grade', 'A')

print(student)  # {'name': 'Alice'}

🔹 pop("age") removes "age" and returns its value.
🔹 popitem() removes and returns the last inserted key-value pair.

5. Looping Through Dictionaries

We can iterate over dictionary keys, values, or key-value pairs.

Example:

student = {"name": "Alice", "age": 21, "grade": "A"}

# Loop through keys
for key in student:
    print(key, ":", student[key])

# Loop through key-value pairs
for key, value in student.items():
    print(f"{key} -> {value}")

Output:

name : Alice
age : 21
grade : A
name -> Alice
age -> 21
grade -> A

🔹 for key in student: iterates over keys.
🔹 for key, value in student.items(): iterates over key-value pairs.

6. Dictionary Comprehensions

Python allows creating dictionaries using dictionary comprehension.

Example:

squares = {x: x**2 for x in range(1, 6)}
print(squares)

Output:

{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

🔹 {x: x**2 for x in range(1, 6)} generates a dictionary of squares.

7. Nested Dictionaries

Dictionaries can contain other dictionaries as values.

Example:

students = {
    "Alice": {"age": 21, "grade": "A"},
    "Bob": {"age": 22, "grade": "B"},
}

print(students["Alice"]["grade"])  # A

🔹 students["Alice"]["grade"] returns "A" from nested dictionaries.

8. Using defaultdict for Missing Keys

defaultdict (from collections module) avoids KeyError by providing a default value for missing keys.

Example:

from collections import defaultdict

# Default value for missing keys is an empty list
scores = defaultdict(list)

scores["Alice"].append(95)
scores["Bob"].append(88)

print(scores)

Output:
defaultdict(<class 'list'>, {'Alice': [95], 'Bob': [88]})

🔹 defaultdict(list) initializes missing keys with an empty list.
